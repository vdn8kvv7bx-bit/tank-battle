# 🎮 坦克大战 (Tank Battle)

经典 FC 红白机坦克大战的 HTML5 复刻版。一个纯前端实现的射击游戏，无需任何依赖。

## 🎯 游戏特色

- 🕹️ **经典玩法** — 控制坦克，保卫基地，消灭所有敌军
- 🧱 **可破坏砖墙** — 射击砖墙可以开辟新路径
- 🛡️ **钢铁墙壁** — 无法摧毁的障碍，需要绕路
- 🌊 **水域地形** — 坦克无法通过，但子弹可以穿越
- 🤖 **多种敌人** — 普通坦克、快速坦克、重型坦克（2点生命）
- 🎁 **道具系统** — 随机掉落：射速提升⭐、无敌护盾🛡、冰冻❄、全屏炸弹💣
- 📈 **难度递增** — 每关敌军更多，同时在场数量增加
- 🎨 **像素风格** — 复古像素美学，忠实还原经典

## 🎮 操作方式

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` 或 `W` `A` `S` `D` | 移动坦克 |
| `空格` | 发射子弹 |
| `P` | 暂停/继续 |

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏！

```bash
# 克隆仓库
git clone https://github.com/vdn8kvv7bx-bit/tank-battle.git

# 打开游戏
cd tank-battle
open index.html  # macOS
# 或直接双击 index.html 在浏览器中打开
```

## 🛠️ 技术栈

- 纯 HTML5 / CSS3 / JavaScript
- Canvas 2D 渲染
- 无框架、无依赖

## 📝 游戏规则

1. **目标**：消灭关卡中所有敌军坦克，同时保护基地（🦅）
2. **生命**：初始 3 条命，被击中后短暂无敌
3. **胜利**：消灭所有 20 辆敌军坦克
4. **失败**：生命耗尽 或 基地被摧毁

## 🎁 道具说明

| 道具 | 效果 |
|------|------|
| ⭐ 星星 | 射速翻倍（持续 10 秒） |
| 🛡 护盾 | 无敌状态（持续 3 秒） |
| ❄ 雪花 | 冰冻所有敌军（持续 5 秒） |
| 💣 炸弹 | 立即消灭场上所有敌军 |
